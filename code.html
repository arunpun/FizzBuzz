The Code for APP


                        //Start function
function getValues() {
    //Get values from the page
    let fizzValue = document.getElementById("fizzValue").value;
    let buzzValue = document.getElementById("buzzValue").value;

    //Type casting
    fizzValue = parseInt(fizzValue);
    buzzValue = parseInt(buzzValue);

    //Validation
    if (Number.isInteger(fizzValue) && Number.isInteger(buzzValue)) {
        let fbArray = fizzBuzz(fizzValue, buzzValue);

        //Display data
        displayData(fbArray);
    } else {
        alert("Error: You must enter integers!");
    }
}

//Number or fizz or buzz?
function fizzBuzz(fizzValue, buzzValue) {
    let returnArray = [];

    //Loop from 1 to 100
    for (let i = 1; i <= 100; i++) {
        //Check to see if divisible by both
        if (i % fizzValue == 0 && i % buzzValue == 0) {
            returnArray.push("FizzBuzz");
            //Check to see if divisible by fizz value
        } else if (i % fizzValue == 0) {
            returnArray.push("Fizz");
            //Check to see if divisible by buzz value
        } else if (i % buzzValue == 0) {
            returnArray.push("Buzz");
        } else {
            returnArray.push(i);
        }
    }

    return returnArray;
}

The code is structured in three functions, getValues(), fizzBuzz() & displayData()

getValues()

The function is triggered by an event listener, document.getElementById("btnSubmit").addEventListener("click", getValues); The function then creates two variables, fizzValue & buzzValue whose values are passed respectively with the document.getElementById().value; method which are then converted into integers with the pareseInt() method. Inside a for loop, these parsed variables are then verified with the Number.isInteger() method. If both of the expression evaluates to true, the function creates an array called fbArray which is used to call the fizzBuzz() function, which takes the variables, fizzValue & buzzValue as parameters. Since the function fizzBuzz() will return an array, the fbArray is further used as an argument to the displayData() function. If there are any invalid data, the user is presented with an appropriate message.

fizzBuzz()

This function first creates an empty array, returnArray, which will hold the final results. Since we already have access to fizzValue & buzzValue, we can commence the logic of this classic coding challenge in a for loop. The loop starts with an starting index of 1 which goes up to 100 while incrementing the index by 1 on each loop. Next, we have an if, else if & else statements. The statement first checks if the current value at i is cleanly divisible by both of the fizzValue & buzzValue. If the expression evaluates to True, the string "FizzBuzz" is pushed to the returnArray. Next up, we have an else if statement which checks if the current value at i is cleanly divisible by the fizzValue. If that is the case, the string Fizz is pushed into the return array. Similarly, we have another else if statement to check if the index value is cleanly divisible by the buzzValue. In this case, Buzz is pushed into the return array. Lastly, if none of the above statements are satisfied, the loop pushes the number itself into the array. The function then returns the results in an array.

displayData()

This function takes in a parameter, fbArray and initializes an empty string called templateRows. Using the array.forEach() method, the function starts to loop through each element in the argument array. Here, I have implemented a simple if/else statement to check whether a given element in the array is a number or a string. If the element is an integer, using template literals, the function creates tr and tdtemplateRows variable.
